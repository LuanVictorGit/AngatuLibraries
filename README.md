## 📌 Sobre

O **AngatuLibraries** é um conjunto de utilidades criado para atender às necessidades internas da **Angatu Sistemas**, com foco em produtividade, organização e redução de código repetitivo.

A biblioteca centraliza soluções comuns utilizadas em múltiplos projetos, permitindo maior consistência e manutenção simplificada.

---

## 🚀 Objetivo

Fornecer uma base sólida com funcionalidades reutilizáveis, incluindo:

* Execução e gerenciamento de tarefas (assíncronas, com delay e recorrentes)
* Manipulação de dados
* Utilitários gerais (strings, datas, validações)
* Helpers para desenvolvimento backend
* Estruturas auxiliares para projetos escaláveis

---

## 📦 Instalação

### Maven (via JitPack)

Adicione o repositório:

```xml id="m0zq2k"
<repositories>
    <repository>
        <id>jitpack.io</id>
        <url>https://jitpack.io</url>
    </repository>
</repositories>
```

Adicione a dependência:

```xml id="4d7jhe"
<dependency>
    <groupId>com.github.LuanVictorGit</groupId>
    <artifactId>AngatuLibraries</artifactId>
    <version>VERSION</version>
</dependency>
```

> Substitua `VERSION` pela versão desejada disponível no JitPack.
